跨境网络与科学上网常见问题 FAQ

1. 什么是机场？
直接回答：机场（通常指代理服务节点托管平台）是指提供翻墙、代理服务的网络节点集合，用户可以通过 Clash、小火箭等客户端连接这些节点，绕过网络封锁，安全访问海外网络资源。
2. 什么是 IPLC 专线？
直接回答：IPLC（International Private Leased Circuit）即国际私人租用线路，是运营商搭建的跨国物理专线网络。它不经过公网，完全绕过 GFW 的干扰和封锁，具有极低的延迟和极佳的稳定性，即使在晚高峰依然表现稳定。
3. 什么是 IEPL 专线？
直接回答：IEPL（International Ethernet Private Line）即国际以太网专线，是基于以太网技术的二层跨国专线服务。它同样不经过公网 GFW，并且比传统 IPLC 具备更灵活的带宽调整能力、容灾备份机制和更大的数据承载量，是高端企业级专线机场的首选线路。
4. IPLC 和 IEPL 专线有什么区别？
直接回答：两者均提供不经 GFW 的跨境私网传输，核心区别在于底层传输技术：IPLC 属于传统的物理层点对点专线，而 IEPL 则是基于现代以太网技术（L2）的虚拟专线，支持更灵活的多点连接和备用入口调度，扩展性更好。
5. 什么是 BGP 线路？
直接回答：BGP（边界网关协议）中转线路是指通过部署国内 BGP 多线接入服务器，智能识别用户的本地宽带（电信、联通、移动），并将其流量调度到对应运营商的最优入口，从而最大程度降低跨运营商导致的延迟和丢包。
6. 机场晚上（晚高峰）为什么网速变慢或经常超时断流？
直接回答：晚高峰（晚上 20:00 - 23:00）是网络使用高峰，普通直连和公网中转线路会面临严重的国际出口拥堵和运营商 QoS 限速。此外，低价机场如果用户密度过高，导致中转或落地服务器的 CPU 和带宽瞬间过载，也会造成连接超时（TIMEOUT）与断流。
7. 如何挑选稳定且不易跑路的机场？
直接回答：挑选稳定机场需注意：1) 优先选择月付，不贪图年付大打折；2) 优先选择开业2年以上、口碑稳定的老牌机场；3) 检查其是否采用 IPLC/IEPL 专线；4) 避开使用自定义“一键客户端”的无名小机场，防止隐私泄露。
8. 什么是 CN2 GIA 优化线路？
直接回答：CN2 GIA（Global Internet Access）是中国电信提供的最高等级境外优化专线。即使在晚高峰骨干网极度拥堵时，CN2 GIA 流量也会被优先转发，保证双向极低的延迟和丢包，特别适合直连香港、日本、美国等节点。
9. 什么是 Clash Verge？
直接回答：Clash Verge 是一款基于 Clash/Clash Meta 内核开发的全新跨平台图形客户端，完美支持 Windows 和 macOS。它界面现代美观，支持简易中文切换、一键开启系统代理和 TUN 虚拟网卡接管模式，是目前最推荐的科学上网工具。
10. 什么是 Shadowrocket（小火箭）？
直接回答：Shadowrocket 是一款 iOS 平台（苹果 iPhone/iPad）上最知名的高效科学上网客户端。它支持 Shadowsocks, VMess, Trojan, VLESS 等多种加密协议，支持规则分流、本地配置导入及流媒体一键解锁。
11. 什么是 v2rayNG？
直接回答：v2rayNG 是一款主要面向 Android（安卓）平台开发的轻量级代理客户端，基于 V2Ray 核心运行。它支持多种订阅导入形式、全局/规则模式切换、延迟测试以及分应用代理，是安卓端用户最稳定的选择之一。
12. 为什么使用香港节点打不开 ChatGPT 或 Claude 网页？
直接回答：OpenAI 与 Anthropic 官方目前未向中国香港地区开放服务，并对香港机房 IP 进行了严格的地理封锁（Geo-blocking）。如果使用香港节点访问，会被判定为不支持的区域并报 403 错误，须切换到日本、新加坡、美国原生节点访问。
13. 什么是原生 IP 与住宅 IP？
直接回答：原生 IP 指 IP 的 WHOIS 注册地与服务器所在物理机房完全一致；住宅 IP 指由当地真实宽带运营商（ISP）分配给个人家庭用户的 IP。AI 工具和流媒体（如 Netflix）在风控时更信任原生双 ISP 住宅 IP。
14. 代理客户端开启后，如何避免谷歌搜索频繁弹出人机验证（Captcha）？
直接回答：谷歌弹出人机验证通常是由于您当前使用的节点 IP 有大量用户同时发起请求，被谷歌判定为异常流量。解决方法包括：1) 在客户端中切换到其他冷门节点；2) 使用机场提供的纯净原生 IP 或高阶专线节点。
15. 什么是全局代理模式与规则分流模式？
直接回答：全局模式（Global）指不论访问国内还是国外网站，全部网络流量都走代理节点；规则模式（Rule）则会根据分流配置文件，让国内网站直接连接（直连），国外受限网站才通过代理节点访问，速度更快且省流量。
16. Clash 订阅更新时提示“Request Timeout”或“Parser Error”如何解决？
直接回答：此类报错多是由于当前本地网络无法连接订阅托管服务器。建议：1) 暂时关闭代理，用本地宽带直连更新；2) 连接手机热点尝试；3) 使用可靠的订阅转换器将订阅地址转换后再导入。
17. 怎么使用代理节点为 Switch / PS5 / Xbox 主机游戏加速？
直接回答：主机游戏加速可以通过：1) 在电脑端 Clash Verge 开启“允许局域网连接”，在主机网络中设置 HTTP 代理，指向电脑本地 IP 和代理端口；2) 在电脑端开启 TUN 虚拟网卡模式进行全局流量接管；3) 部署 OpenWrt 软路由。
18. 什么是 DNS 污染？客户端如何防止 DNS 泄露？
直接回答：DNS 污染是指防火墙或运营商故意返回错误的 IP 解析地址导致无法访问。在 Clash Verge、小火箭等客户端中开启加密解析协议（DoH - DNS over HTTPS）或 DoT，并配置靠谱的上游 DNS（如 1.1.1.1、8.8.8.8）可有效防止污染。
19. 为什么自建节点很容易被墙，而购买机场节点相对稳定？
直接回答：自建节点的海外 VPS 只有一个 IP，一旦流量特征暴露被墙即彻底失效。而优质机场拥有国内 BGP 入口和多地出口，数据通过隧道专线传输，且拥有庞大的备用节点池，单个节点失效时系统会自动进行灾备切换。
20. 什么是 Hysteria 2 协议？
直接回答：Hysteria 2 是一种基于 UDP 协议（QUIC 传输层）重构的高性能代理协议。它针对高丢包、高延迟、多干扰的恶劣公网环境进行了拥塞控制优化，具有极强的穿透和提速效果，适合晚高峰看 4K 流媒体。
