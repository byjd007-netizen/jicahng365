Mac Stash / Clash Verge 使用教程（macOS 完整配置指南）

作者: 技术编辑部 阅读时长: 约 10 分钟 发布于: 2026-06-22 更新于: 2026-06-22
客户端教程 科学上网 网络科普 阅读量: 12.5k
在现代网络环境中，“代理”已成为连接全球互联网的重要技术手段。了解代理的原理与常见协议，是提升网络体验的关键一步。

一、Stash / Clash Verge 是什么？

在 macOS 上，代理工具主要分为两大类：

1. Clash Verge

Clash Verge 是基于 Clash 核心开发的现代化客户端，特点是：

  • 界面简洁直观
  • 支持多协议节点
  • 支持规则分流（Rule Mode）
  • 支持订阅自动更新
  • 适合新手使用

2. Stash

Stash 是 iOS / macOS 上非常强大的代理工具，特点是：

  • 功能更高级
  • 支持复杂规则系统
  • 支持多种策略组
  • 更适合进阶用户

二、下载安装（Mac）

1. Clash Verge 下载

建议通过官方 GitHub 获取最新版：

搜索 Clash Verge macOS 版本（DMG安装包） 获取下载链接。

安装步骤：

  1. 下载 DMG 文件
  2. 打开安装包
  3. 拖动图标到 Applications（应用程序）文件夹
  4. 打开应用

2. Stash 下载

Stash 通常通过 App Store 或 TestFlight 获取：

步骤：

  1. 打开 Mac 上的 App Store
  2. 搜索 Stash
  3. 购买、下载并安装

三、首次启动基础设置

1. 语言设置

进入 Settings（设置）：

  • 选择 Language -> 中文（简体）

2. 开机启动（推荐）

在设置中开启：

  • Launch at startup（开机启动）

3. 权限设置

macOS 首次运行可能会提示权限请求：

  • 网络权限
  • VPN 权限

四、导入订阅节点（核心步骤）

这是整个配置过程中最关键的一步。

方法一：URL导入（推荐 Clash Verge）

Clash Verge 操作步骤：

  1. 打开 Clash Verge
  2. 点击左侧 Profiles（配置）
  3. 点击 “+” 添加配置
  4. 选择 URL 导入
  5. 粘贴订阅链接
  6. 点击确认

等待几秒即可自动加载节点列表。

方法二：Stash 导入订阅

操作步骤：

  1. 打开 Stash
  2. 点击 Profiles（配置）
  3. 选择 Add Subscription（添加订阅）
  4. 输入订阅链接
  5. 保存并更新

五、开启代理与节点选择

1. 选择节点

进入 Proxy（代理列表），常见节点选择：

  • 香港（低延迟）
  • 日本（稳定）
  • 美国（流媒体）

2. 开启系统代理

在主界面开启：System Proxy（系统代理）。

六、模式说明（非常重要）

1. Rule 模式（推荐）

智能分流：国内网站直连，国外网站走代理。

日常使用最稳定，推荐开启。

2. Global 模式

所有流量强制走代理。

用于：测试网络、或特定全局代理场景。

3. Direct 模式

关闭代理，所有流量直连。

用于：排查问题、恢复正常的本地网络。

七、Stash 高级功能说明

对于使用 Stash 的进阶用户：

1. 策略组（Policy Group）

可以自定义：自动选择最快节点、手动切换节点、按区域分组等。

2. 规则系统（Rules）

可以实现复杂的定制：广告屏蔽、国内直连、指定网站走特定代理通道等。

3. 延迟测试

测试节点速度，以便精准选择最优线路。

八、常见问题解决

1. 订阅导入失败？

  • 可能原因：链接过期、网络不通、复制错误
  • 解决：更新订阅、更换网络重新尝试

2. 节点无法连接？

  • 原因：节点失效、配置错误
  • 解决：更换其他节点、更新订阅

3. VPN连接但无法上网？

  • 检查：是否开启系统代理、是否选中有效节点、是否处于 Rule 模式

九、使用建议（新手必看）

十、总结

在 Mac 上使用 Stash 或 Clash Verge，可以轻松实现稳定的网络代理与分流功能。

只需要三步：安装客户端 -> 导入订阅 -> 选择节点并开启代理，即可正常享受全球网络访问。
