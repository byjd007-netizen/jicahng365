iOS Shadowrocket 使用教程（小火箭详细配置指南）

在 iPhone / iPad 上使用 Shadowrocket（俗称“小火箭”）是目前最常见的网络代理方式之一，它可以帮助用户快速导入订阅节点，并实现规则分流、流媒体解锁以及稳定的网络访问体验。

本教程将一步一步教你如何在 iOS 设备上安装 Shadowrocket，并完成节点配置与订阅导入，即使是新手也能轻松上手。

一、Shadowrocket 是什么？

Shadowrocket 是 iOS 平台上一款强大的网络代理工具，它支持多种协议，包括：

  • Shadowsocks（SS）
  • VMess
  • Trojan
  • HTTP / HTTPS 代理
它的核心作用是：

通过配置节点，实现网络请求的分流与加密访问。

简单来说就是：
  • 国内网站正常访问
  • 国外网站通过代理加速访问
  • 支持规则自动分流

二、Shadowrocket 获取方式

注意：Shadowrocket 需要在 App Store 购买下载（部分地区不可见）。

获取方式：

  1. 切换 Apple ID 地区（如美区 / 港区）
  2. 打开 App Store
  3. 搜索：Shadowrocket
  4. 购买并下载

安装完成后，桌面会出现“小火箭”图标。

三、如何添加订阅节点（核心步骤）

Shadowrocket 的核心使用方式是：导入订阅链接

什么是订阅链接？ 订阅链接是机场提供的节点集合地址，一般类似：
https://example.com/api/v1/subscribe?token=xxxxxx
它包含多个节点，例如：香港节点、日本节点、美国节点等。

方法一：一键订阅导入（推荐）

操作步骤：

  1. 打开 Shadowrocket
  2. 点击底部菜单 “配置”
  3. 点击右上角 “+”
  4. 选择 “URL”
  5. 粘贴你的订阅链接
  6. 点击 完成

等待几秒后，系统会自动拉取节点列表。

方法二：手动添加节点

如果没有订阅链接，可以手动添加：

  1. 点击 “配置”
  2. 点击 “添加节点”
  3. 选择协议类型（SS / Trojan / VMess）
  4. 填写服务器信息：地址、端口、密码、加密方式
  5. 保存

四、开启代理与使用方法

节点导入成功后，还需要开启代理。

1. 选择节点

进入 首页 -> 代理列表

选择一个节点，例如：

  • 香港（低延迟）
  • 日本（稳定）
  • 美国（流媒体）

2. 开启连接

打开首页顶部开关：“未连接 -> 已连接”

成功后，手机顶部状态栏会出现 VPN 图标。

五、规则模式说明（非常重要）

Shadowrocket 支持三种模式：

1. 规则模式（推荐）

自动分流：国内网站直连，国外网站代理。最适合日常使用。

2. 全局模式

所有流量走代理。

适合：测试网络、特殊环境使用。

缺点：耗电快、国内速度可能下降。

3. 直连模式

关闭代理。

用于：排查问题、恢复正常网络。

六、常见问题解决

1. 订阅导入失败？

  • 可能原因：订阅链接过期、网络无法访问服务器、链接复制错误
  • 解决方法：重新复制订阅链接、更换 WiFi / 手机流量、检查机场状态

2. 节点无法连接？

  • 可能原因：节点已失效、网络不稳定
  • 解决方法：切换其他节点、更新订阅

3. VPN 已连接但无法上网？

  • 检查：是否选择节点、是否开启代理、是否使用规则模式

4. App 无法下载？

  • 说明：Apple ID 地区不支持、App Store 搜索不到
  • 解决：切换美区/港区 Apple ID

七、进阶使用技巧（提升体验）

1. 延迟测速

点击节点 -> 测试延迟，选择延迟最低节点。

2. 分组管理

可以将节点分组：游戏节点、流媒体节点、ChatGPT节点等。

3. 自动切换节点

开启：“自动选择最快节点”，系统会自动帮你切换最优线路。

八、使用建议（新手必看）

为了稳定使用建议：
  • 不要频繁切换节点
  • 优先使用 Rule 模式
  • 每周更新一次订阅
  • 选择低延迟节点
  • 避免多个 VPN 同时开启

九、总结

Shadowrocket 是 iOS 上最强大的代理工具之一，适合长期稳定使用。

只需要完成三步：下载 Shadowrocket -> 导入订阅链接 -> 选择节点并开启代理，即可实现稳定网络访问。
